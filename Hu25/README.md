# Hu25

FAIR apatite U-Pb data for "Prolonged evolution of an intra-oceanic island arc: The ~60 Myr record of Central Cuba" by Hu et al. (in review)

## Contents

[data](data): raw ICP-MS data generated by an Agilent 7900 instrument at University College London

[json](json): `json` files with post-processing instructions for [`IsoplotR`](https://isoplotr.es.ucl.ac.uk)

[process.jl](process.jl): a `Julia` script that turns `data` into [`json/all.json`](json/all.json)

## Prerequisites

[`Julia`](https://julialang.org): a scientific programming language

[`KJ.jl`](https://github.com/pvermees/KJ.jl): an LA-ICP-MS data reduction package

[`IsoplotR`](https://isoplotr.es.ucl.ac.uk/home/index.html): an `R` package for geochronological data processing

## Installation instructions

At the `Julia` REPL:

```
import Pkg; Pkg.add("https://github.com/pvermees/KJ.jl")
```

The contents of `json` were generated using this particular version of `KJ.jl`:

```
import Pkg; Pkg.add(url="https://github.com/pvermees/KJ.jl#4fba4a2")
```

## FAIR workflow

1. Navigate to the `Hu25` folder and run the following code at the Julia REPL:

```
import KJ
include("process.jl")
```

2. Open the content of `json` in [`IsoplotR`](https://isoplotr.es.ucl.ac.uk)
